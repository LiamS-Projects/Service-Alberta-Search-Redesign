# Service-Alberta-Search-Redesign

A polished, production-style front-end redesign of the Service Alberta Search page, ready to publish with GitHub Pages.

## Run locally

Open `index.html` directly in your browser.

## Deploy on GitHub Pages

This repository includes a GitHub Actions workflow at `.github/workflows/deploy-pages.yml` that deploys the site to GitHub Pages from the `main` branch.

1. Push this repository to GitHub.
2. In **Settings → Pages**, set **Source** to **GitHub Actions**.
3. Push to `main` (or run the workflow manually from the **Actions** tab).
4. Your site will be available at:
   - `https://<your-username>.github.io/Service-Alberta-Search-Redesign/`
   - or your repository-specific Pages URL shown in the workflow run output.
